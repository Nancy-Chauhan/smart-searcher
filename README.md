# Smart Searcher 🔎

👧🏻 SHEHACK-2019 Hackathon Project 🛠️

## Pre-requisites

1. Python 3.7 🐍
2. Pipenv

## Getting Started

1. 🖥️ Launch `pipenv shell` to start-up a virtual environment
2. ⚡ Run `pipenv install` to install required dependencies

## Starting the server

1. ⚡ Run `./server.sh dev` to start a development server
